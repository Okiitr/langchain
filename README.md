# Baby Names Generator

AI Generates cool baby names genertor based on the zodiac sign for you!

![Beby Name Web app demo](/images/baby_name_generator.png)

```bash
    python -m venv .venv
    source .venv/bin/activate
    pip install -r requirements.txt
    streamlit run main-baby-name-generator.py  
```
